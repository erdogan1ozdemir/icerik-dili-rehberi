# Terminoloji ve Sözlük

## 1. Terminoloji Politikası

### 1.1. [A] Kurumsal: İngilizce terim korunur

Sektörde İngilizce kullanılan terim çevrilmez. Türkçe ek alarak çekimlenebilir ("click'ler", "impression'ları", "carousel'ler") - bu melez kullanım kabul edilen ev üslubudur.

**Korunan terim aileleri:**
- Metrikler: `Click, Impression, CTR, Session, Transaction, Revenue, AOV, CR, ECR, Bounce Rate, Avg. Position, Visibility Score, Share of Voice (SoV), Prompt Coverage`
- Dönemler: `YoY, MoM, QoQ, YTD`
- SEO/GEO: `Query, Keyword, Citation, Mention, Brand / Non-Brand, Intent, Long-tail, Featured Snippet, PAA, AI Overview, AI Mode, Zero-click, Fan-out Query, Knowledge Panel, Content Gap, Crawl Budget, Internal Linking, Backlink, Anchor Text, Above the Fold, Market Cluster`
- Teknik: `E-E-A-T, Schema markup / JSON-LD, SSR, SERP, KD, DR, LCP, CLS, INP, CWV, GSC, GA4, H1, Canonical, Indexability, llms.txt, robots.txt, sitemap.xml, machine-readable, Breadcrumb, FAQPage Schema, GMB`

**Zorunlu karşılıklar (Türkçesi kullanılmaz):**

| Kullanma | Kullan |
|---|---|
| atıf | **mention** |
| çapa / bağlantı metni | **anchor text** |
| sorgu (terim bağlamında) | **keyword / query** |
| makine okunur | **machine-readable** |
| üst fold | **above the fold** |
| market kümesi | **Market Cluster** |
| ss. | **session** |
| pik | **peak** |
| Paid talep artışı | **Paid yatırım artışı** |

- Tablo başlıkları ve metrik adları İngilizce kalır; tek tabloda TR/EN karışımı yapılmaz (yorum kolonu Türkçe olabilir).
- Yeni/az bilinen metrikler (SoV, citation, mention, zero-click, agentic commerce) metnin/sunumun başında bir kez sade dille açıklanır; sunumda slayt altı not olarak eklenebilir.
- İstisna: Türkçesi olmayan, sektörde yaygın terimler İngilizce kalır; Türkçesi yaygınsa çevrilir.

### 1.3. Terim tanıtma yöntemi (kanal bazında)

| Kanal | Yöntem |
|---|---|
| Sunum (PPTX) | İlk geçişte `** terim: tek cümlelik tanım` dipnotu veya parantez içi mini karşılık: "query fan-out (sorguyu alt-sorulara bölme)". Birden çok terim aynı satırda "·" ile ayrılabilir. |
| HTML rapor | `<span class="term" data-term="KD">Keyword Difficulty (KD)</span>` - yalnızca glossary'de tanımlı terimler span alır; glossary dışı terim düz yazılır. Tüm işaretli terimler sonda "Terim Sözlüğü"nde toplanır. Rapora özel kısaltmalar (R1/R2/R3 gibi) da aynı yöntemle tooltip'lenir. |
| Word/Excel çıktı | Düz metin + ilk geçişte parantez içi açıklama. |

Karmaşık kavramlar somut senaryoyla açılır: "Agentic commerce nedir? Kullanıcı 'bana 150 doları geçmeyen çadır bul ve al' dediğinde yapay zeka asistanı ürünü kendisi arıyor, karşılaştırıyor ve satın alıyor." ("Basitçe nedir" gibi başlıklar kullanılmaz; "Agentic commerce nedir?" yeterli.)

## 2. Yasak → Tercih Edilen İfadeler (birleşik sözlük)

### 2.1. Kesin vaat

| Kaçınılacak | Tercih edilen |
|---|---|
| garantiliyoruz | potansiyel taşımaktadır |
| kesin, mutlaka | gözlemlenebilir, değerlendirilebilir |
| %X artış olacak | %X artış fırsatı sunabilir |
| hemen, acilen | önceliklendirilebilir |
| 1. sıra çok kolay | 1. sıra hedeflenebilir |
| garanti vermez, garantisi yok (deyim) | beraberinde getirmez / güvence sağlamaz |

### 2.2. Olumsuz / keskin kelime

| Kaçınılacak | Tercih edilen |
|---|---|
| kötü, berbat | geliştirme alanı barındıran |
| yanlış | gözden geçirilebilir |
| başarısız | hedeflenen sonuca ulaşmayan |
| hata, sorun (yargı olarak) | ele alınabilecek nokta / bulgu / tespit |
| ciddi | dikkate değer |
| zayıf | belirli alanlarda büyüme potansiyeli taşıyan |
| en zayıf halka | en az işlenen / en sınırlı kalan başlık |
| ters profil, ters profildedir | tersine dönen örüntü / tam tersi profil |
| yapmıyor, eksik | henüz değerlendirilmemiş / katalogda henüz yer almıyor |

### 2.3. Pazarlama jargonu / abartı

| Kaçınılacak | Tercih edilen |
|---|---|
| patlama (yaşıyor) | yüksek artış, belirgin büyüme |
| büyüme motoru | büyüme oranı / büyüme kanalı |
| dramatik | belirgin, kayda değer |
| açık ara (önde) | belirgin biçimde / net biçimde (önde) |
| kat kat, X katı | somut oran (~%30 daha yüksek) veya "Nx" çarpan (3.6x) |
| puan iyileşme | pozisyon iyileşmesi / sıralama iyileşmesi |
| para tuzağı | ek süre maliyeti (nötr betimleme) |
| cannibalization | fırsat kaybı |
| maksimum verimlilik, hayat veriyor | somut, ölçülü karşılık |

### 2.4. Teknik bulgu kalıpları (kelime düzeyi)

| Kullanma | Kullan |
|---|---|
| "Sorun: görsellerde boyut yok" | "Hero görsellerinde boyut tanımı bulunmamaktadır" |
| "Bu CLS'i bozuyor" | "Bu durum CLS değerini olumsuz yönde etkilemektedir" |
| "Hemen düzeltin" | "İlgili düzenlemenin kısa vadede yapılması tavsiye edilir" |
| "CSS'i inline yerleştirin" | "Kritik CSS inline yerleştirilebilir mi?" |
| "Kullanılmayan CSS'i kaldırın" | "Kullanılmayan CSS kurallarının temizlenmesi önerilir" |
| "Font'u preload ile yükleyin" | "İlgili font dosyası preload ile ön yüklenebilir mi?" |

Not: Gözlemlenen/ölçülmüş etkiler "-mektedir / -maktadır" ile net bildirilir ("etkilemektedir", "bulunmamaktadır"); "-ebilmektedir" yalnızca etkinin kendisi belirsizse kullanılır. Yumuşatma öneri katmanına aittir, tespit katmanına değil.

### 2.5. Meta yorum ve iddia yumuşatma

- "Analizi yeniden çerçeveledik" → doğrudan olgu: "Ahrefs'te 5G %2.3 görünüyordu; gerçekte en büyük paid segment olduğu tespit edildi."
- "Tüm durumlar tek tek doğrulandı" → "mevcut durum kontrol edilerek ilerlendi."
- "Sentez / doğrulandı" ibareleri → kaldırılır; varsa kaynak/referans linki verilir.

### 2.6. Çiğ / coined kelime ve etiket dili

- Uydurma / coined kelime kullanılmaz: "Duallik" → "İkili sorgu"; "havlu paradoksu" → nötr başlık.
- Grafik/tablo etiketleri, eksen adları, cluster & segment adları da gövde metniyle aynı dil denetimine tabidir; "küçük etiket" diye atlanmaz.
- Sabit sektör terimi olmayan ham İngilizce ifadeler Türkçeleştirilir: "weak topic clusters" → "düşük görünürlüklü cluster'lar". Yalnızca yerleşik terimler (SOV, CTR, citation, mention, cluster) İngilizce kalır.

## 3. Rozet / Etiket Eşlemeleri

### 3.1. Önceliklendirme dili

- Aksiyon planı gün/hafta değil **"Öncelik 1/2/3"** veya **"Faz 1/2/3"** ile verilir.
- Öncelik dereceleri Türkçe ve sade: Öncelikli / Yüksek / Orta / Takip edilebilir.
- Proje durum etiketleri şeffaf: "Devam Ediyor / Hazırlık Aşaması / Planlandı".

### 3.2. Sosyal dinleme / algı raporu ek eşlemesi [B]

| Ham etiket | Raporda |
|---|---|
| Negative | **Olumsuz** (nötr betimle) |
| Neutral / mixed | **Nötr / Karışık** |
| Positive | **Olumlu** |
| Escape / churn intent | **Gitme niyeti** |
| Crisis / risk | **İzlenmeli / dikkat** |
| Unverified | **Doğrulanamadı** (şeffaf rozet) |
| Verified | **Doğrulandı** |
| Recent | **Güncel / taze** (tarihiyle) |
| AI-generated metric | **AI üretimi (temkinli)** |

### 3.3. Fırsat raporu etiketleri (Turkcell tipi)

| Ham | Raporda |
|---|---|
| Existing / Live | **(MEVCUT)** |
| Example / long-tail | **(*)** |
| Approved by brand | **★★★ MARKAYA İLETİLMİŞ** |
| Discussed in meeting | **(konuşuldu)** |
| Priority high/med/low | **★★★ / ★★ / ★** |
